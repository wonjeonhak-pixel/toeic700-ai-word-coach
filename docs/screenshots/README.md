## 撮影前のチェック

- [ ] ハードリロード後の最新本番ビルドを使用
- [ ] DevTools のデバイスエミュレーションを iPhone 14 / Pixel 7 相当（幅 390〜412px）に
- [ ] ブラウザの UI Chrome を最小化（フルスクリーンや「スクリーンショット」モード）
- [ ] ダーク UI 拡張 / カラー反転を無効化
- [ ] 画面下部の `QuietStep` signature が見える場合はそのまま含めて OK

## 形式

- PNG 推奨（透過不要、UI そのまま）
- 1 枚あたり 500KB 以下を目安に、可能なら最大幅 720px に縮小
- ファイル名は上記表のとおり（番号付きで並び順を担保）

画像追加後は本ファイルの表を更新するか、そのまま使えます。
